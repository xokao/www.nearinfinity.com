Speaker: Seth Schroeder, Senior Software Developer, Near Infinity
Usergroup: National Capital Area Clojure Users Group (a.k.a. Cap-Clug)

  • Reston, Virginia, Jan 14

Clojure is an exciting, new dynamic programming language for the Java Virtual Machine. A vibrant community has sprung up around Clojure and started to produce infrastructure.

Leiningen is a Clojure-friendly build tool initiated and led by Phil Hagelberg. It provides a way to create skeleton applications, and eagerly supports fleshing them out with any dependency Maven can track. Testing and packaging code is supported out of the box. Let Leiningen handle project-common tasks so you can focus on what your project specifically needs!
