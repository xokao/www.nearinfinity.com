NIC Website
===========


QuickStart
----------

This will run a server on locahost that regenerates any changes.

    > bundle install
    > bundle exec jekyll --server --auto

Server starts on <http://localhost:4000>


Git Submodule Stuff
-------------------

    > git submodule init
    > git submodule update
    > cd blogs
    > git checkout master
    > git pull
