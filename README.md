NIC Website
===========


QuickStart
----------

    > git clone --recursive git@github.com:nearinfinity/nearinfinity.github.com.git
    > bundle install
    > bundle exec jekyll --server --auto

This will run a server on <http://localhost:4000> that regenerates any changes


Git Submodule Stuff
-------------------

    > cd blogs
    > git checkout master
    > git pull
